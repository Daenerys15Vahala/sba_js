SBA JAVASCRIPT FUNDAMENTALS

This projects includes fundamental JavaScript concepts by analyzing and transforming learner assignment data. It processes course, assignment and student submissions information, to calculate aerages, scores, errors and validating data.

The program takes in course information, assignment groups, and learner submissions, then returns an organized array of learner objects containing:

* Learner ID
* Assignment scores as percentages
* Weighted average score

The program also handles different scenarios as:

* Invalid course and assignment group relationships
* Assignments that are not due yet
* Invalid point values
* Late submissions with score deductions

Used
* JavaScript
* Node.js
* Git/GitHub

This project demonstrates the use of:

* Functions
* Parameters and return values
* Arrays and objects
* Array methods:
* Loops
* Conditional statements
* Error handling with `try/catch`
* Date objects
* Object manipulation
